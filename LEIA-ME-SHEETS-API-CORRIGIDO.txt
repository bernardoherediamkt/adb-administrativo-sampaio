ADB Administrativo Multi-Igrejas - Sheets API Corrigido

Correções desta versão:
- Quando o assunto envolve planilhas, financeiro, membros ou cadastro, o Adam usa Google Sheets API para ler o conteúdo.
- Drive API fica apenas como localizador de IDs/metadados de planilhas dentro da pasta-mãe da igreja selecionada.
- Perguntas financeiras não exportam Google Docs nem leem arquivos genéricos.
- Perguntas com mês específico leem somente o mês pedido.
  Ex.: “março desse ano em relação a março do ano passado” seleciona março do ano atual e março do ano anterior.
- Quando uma planilha é anual e o mês está em abas internas, o Adam filtra abas pelo mês pedido.
- O Adam reconhece faturamento, receita, arrecadação, recebimento, entradas, dízimos e ofertas como assunto financeiro.
- O Adam respeita churchId/churchName e responde somente sobre a igreja selecionada.
- O Adam não trata o usuário como Pastor Bernardo; usa tratamento genérico ou pergunta o nome.
- vercel.json configurado com maxDuration para api/adam-chat.js e api/drive-test.js.
- service-worker atualizado e não intercepta /api/*.
