ADB Administrativo — Pacote corrigido para Vercel

Suba TODOS estes arquivos/pastas na raiz do repositório GitHub:

- index.html
- package.json
- vercel.json
- manifest.webmanifest
- service-worker.js
- icons/
- api/
- scripts/

No Vercel:
- Application Preset: Other
- Root Directory: ./ se os arquivos estiverem na raiz
- Environment Variables:
  Key: GEMINI_API_KEY
  Value: sua chave nova do Google AI Studio

Opcional:
  Key: GEMINI_IMAGE_MODEL
  Value: gemini-3.1-flash-image

Se os arquivos ficarem dentro de uma pasta, configure Root Directory com o nome da pasta.
