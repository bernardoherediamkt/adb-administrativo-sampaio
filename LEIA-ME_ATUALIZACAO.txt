Atualização ADB Administrativo - Controle Financeiro 2026

Para atualizar no GitHub:
1. Envie os arquivos desta pasta para o mesmo repositório.
2. Substitua os arquivos antigos quando o GitHub perguntar.
3. Clique em Commit changes.
4. Aguarde alguns minutos o GitHub Pages atualizar.

Arquivos principais:
- index.html
- manifest.webmanifest
- service-worker.js
- pasta icons

No iPhone, se o app ainda mostrar a versão antiga, feche e abra novamente. Se persistir, remova da tela inicial e adicione novamente pelo Safari.
